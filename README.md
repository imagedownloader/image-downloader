## 🚀 What is Image Downloader?

**Image Downloader** is a powerful, free browser extension and web tool that lets you extract, filter, preview, and bulk-download all images from any webpage in seconds.

No more right-clicking images one by one. Just open the extension on any page, pick the images you want, and download them all — individually or as a single ZIP archive.
